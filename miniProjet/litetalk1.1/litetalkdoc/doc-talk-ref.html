Litetalk sentence syntax

General syntax

SENTENCE = COMMAND PARAM1 PARAM2 .. STRESSERS

COMMAND must be in first position
STRESSER must be at end
PARAMS can be given in any order (except for values: ".... is V")

COMMAND

COMMAND = CODE | NAME | ALTERNATIVE
CODE = a .. z (col 1 of table above)
NAME = ask .. extra (col. 2 aof table above)

STRESSERS

No implemented in V1.0

It is possible to modalize the performatives with some extra stressers
stressers must be at the end of input (possibly followed by whitespaces)
one can give only one stresser by input
"please" can be placed anywhere

Form Description
! makes the performative stronger
? puts and interrogative form to the request
* stands for 'please' meaning politeness or moral demand, ...
@ stand for 'apologize' or 'sorry' politely
& puts an accent of irony to the request just like -:)

These forms can be doubled for more force :

Form Description
!! strong order, strong bravo, strong disagree
?? surprise
** beg
@@ strong apology, excuse, regretsy
&& counterstatement: j P && == j \[Not]P
